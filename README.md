## Descrição

NUKERS Tools é um script bash simples e interativo que facilita a instalação e execução de ferramentas de teste Ddos de rede, como Slowloris, GoldenEye, Hulk, Xerxes e Xspear. Ele fornece um menu para gerenciar essas ferramentas, tornando o processo mais amigável para usuários Linux/Unix.

**Aviso Legal:** Este script é destinado **para fins educacionais** O uso dessas ferramentas para atacar sistemas sem permissão é ilegal. O autor não se responsabiliza por mau uso.

## Pré-requisitos

- **Sistema Operacional:** Linux/Unix-like (testado em Ubuntu/Debian).
- **Dependências:**
  - Bash (padrão na maioria das distros).
  - Python 2/3 (para Slowloris, GoldenEye e Hulk).
  - Git (para clonar repositórios).
  - Compilador C (gcc) para Xerxes.
  - Permissões de root/sudo para instalações (recomendado).

## Uso
``git clone https://github.com/Mtx-rng/Nk-system.git``

``cd Nk-system``

``chmod +x menu.sh``

``./menu.sh``
